Open Source Projects

The following are my open source projects. Please feel free to use them, fork them, love them or hate them.

timer360
timer360 is a jQuery plugin that creates an animated timer. It's based on the iTunes preview animation.
menu_roll
menu_roll is a jQuery plugin that creates an animated menu for use on your website.
content_player
content_player is a jQuery plugin that is my contribution to the mass of slideshow plugins.
